# Real-time-chat-react
Client de chat em tempo real usando react

## Configuracao da API

Crie um arquivo `.env` na raiz do projeto com:

```env
VITE_API_BASE_URL=https://chat-training-api.onrender.com
```

Com esse `.env`, `npm run dev` conecta no servidor em nuvem.

Se a variavel nao existir, o frontend usa esse mesmo endpoint do Render como padrao.
